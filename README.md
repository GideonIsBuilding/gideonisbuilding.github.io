# Gideon Aleonogwe Portfolio

A single-page portfolio site for myself, presented as an "Infrastructure Field Manual".

## Tech Stack
- HTML
- CSS
- JavaScript
- Google Fonts: Bricolage Grotesque, Public Sans, and JetBrains Mono

## Project Structure
- `index.html` - page structure and content
- `styles.css` - all layout, visual design, and responsive styling
- `script.js` - interactive behavior and dynamic content rendering

## Getting Started
This is a static site, so there are no build steps or package installs required.
